# Team Task Manager - Demo Script

**Estimated time: 3-5 minutes**

## 1. Introduction (0:00 - 0:30)
- "Hello, today I'll be demonstrating the Team Task Manager application, a MERN stack project built for efficient team collaboration."
- "The app features robust role-based access control, allowing Admins to manage projects and tasks, while Members can update their progress."
- "Let's start by signing in."

## 2. Authentication & Onboarding (0:30 - 1:00)
- *Action:* Navigate to the Login screen.
- "The authentication system is built using JWT and bcrypt for secure access."
- *Action:* Create a new Admin account or login as an existing Admin.
- "Upon login, we are greeted by the interactive dashboard."

## 3. Dashboard Overview (1:00 - 1:45)
- *Action:* Show the Dashboard.
- "The dashboard gives a bird's-eye view of task distribution. You can see total tasks, how many are completed, pending, or overdue."
- "The pie chart dynamically reflects task statuses, and alerts highlight if any deadlines have been missed."

## 4. Project Management (1:45 - 2:30)
- *Action:* Navigate to the Projects tab.
- "As an Admin, I can create new projects and assign team members."
- *Action:* Click 'New Project', fill out the details, select members, and submit.
- "The project instantly appears, showing the assigned members and the overall deadline."

## 5. Task Management & Workflows (2:30 - 3:30)
- *Action:* Navigate to the Task Board tab.
- "Here is the Task Board. We can filter tasks by project or search for specific items."
- *Action:* Click the '+' button to add a task. Assign it to a project and a specific member.
- "I'll assign this task to a team member and set the deadline."
- *Action:* Change the task's status from 'To Do' to 'In Progress'.
- "Members can update their task statuses, and if a deadline passes without completion, the system automatically flags it as Overdue."

## 6. Conclusion (3:30 - 4:00)
- "This application demonstrates full-stack capabilities, including RESTful API design, database modeling, and a responsive frontend with state management."
- "Thank you for watching the demonstration."
